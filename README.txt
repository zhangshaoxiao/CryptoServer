# CryptoServer
提供加密服务、数据库服务、打log、资瓷并发的服务器框架，不够完善，目前打算套进程池以提升效率，以及附加流量监控功能，正在逐步完善，
很多服务不必要自己造轮子实现，可以借用现在很多的免费云平台，比如即时通讯功能，短信验证功能，各种推送功能以及支付接口等等的。
以下是总结，非原创，转载的，作者信息见下文

5. 即时通讯

在线聊天是一个通用性很强，但又相对难开发的功能，对于很多应用来说聊天只是一个附加的锦上添花的功能（比如很多 O2O 应用的咨询功能），专门投入精力做即时通讯需要消耗大量的开发资源，这时就可以通过嵌入第三方 SDK 的方式来实现在线聊天，只需嵌入几行代码即可实现，市面提供即时通讯的服务有：
